🛒 ProductHub
📌 Project Overview

ProductHub is a web application designed to help users explore and discover products in a simple and efficient way. The application fetches real-time product data from a public API and displays it in a clean and responsive interface.

This project is built as part of a capstone assignment to demonstrate understanding of JavaScript, API integration, and modern UI development.

🎯 Objective

The objective of this project is to:

Build a dynamic web application using JavaScript
Integrate a public API using the fetch method
Display real-time data on a webpage
Implement interactive features like search, filtering, and sorting (in later stages)
🔌 API Used

This project uses the following public API:

👉 https://dummyjson.com/products

The API provides product data such as:

Product title
Description
Price
Category
Images
⚙️ Planned Features

The following features will be implemented in the project:

🔍 Search Functionality
Users will be able to search for products using keywords
🗂 Filtering Options
Products can be filtered based on categories
📊 Sorting Mechanism
Products can be sorted by price or name
❤️ Wishlist / Favorites
Users can save products using localStorage
🌙 Dark Mode / Light Mode
Toggle between themes for better user experience
📱 Responsive Design
The application will work on mobile, tablet, and desktop
